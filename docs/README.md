# Easy Car Luxury - Plataforma de Compra y Venta de Vehículos de Lujo

## Descripción

Easy Car Luxury es una plataforma completa para la compra y venta de vehículos de lujo en Colombia. 
El sistema incluye gestión de usuarios, publicaciones, membresías, pagos integrados con Wompi, 
facturación electrónica con DIAN, y paneles administrativos.

## Tecnologías Utilizadas

- **Backend**: PHP 8.0+ (PDO para base de datos)
- **Frontend**: HTML5, CSS3, JavaScript, Bootstrap 5
- **Base de Datos**: MySQL 5.7+ / MariaDB 10.3+
- **Gráficas**: Chart.js
- **PDF**: mPDF
- **Pagos**: Wompi API
- **Facturación**: DIAN (Facturación Electrónica)
